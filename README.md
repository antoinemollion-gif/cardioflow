# CardioFlow

## Project Overview
CardioFlow is a project designed to help users track and manage their cardiovascular health through a user-friendly interface. It provides insights, analytics, and personalized recommendations.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/antoinemollion-gif/cardioflow.git
   ```
2. Navigate into the project directory:
   ```bash
   cd cardioflow
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Features
- User registration and login
- Dashboard for health metrics overview
- Visual analytics for tracking progress
- Customizable health goals
- Integration with wearable devices
- Notifications and reminders for health check-ups
